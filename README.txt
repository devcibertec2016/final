Esta aplicación se hizo con Mvc 5 usando el IDE de desarrollo Visual Studio 2013, este proyecto esta hecho con Boostrap,
Jquery, este proyecto contiene un mantenimiento(CRUD) completo sobre : 

-> Productos
-> Categorías
-> Proveedores
-> Roles
-> Usuarios

Se hacen validaciones mediante Jquery, cada observación y/o mensaje de error se usan modales y mensajes de confirmación,
También cuenta con un login que mediante el rol que tenga el usuario te muestra los modulos.
-Administrador =  Productos, Categorías, Proveedores, Roles  y Usuarios.
-UsuarioNormal =  Productos, Categorías y Proveedores.

-Cuenta con una prueba unitaria simple de Roles.
